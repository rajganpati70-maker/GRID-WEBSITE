# GRID — ML Community Platform

A React + Vite frontend for an ML research community. Features pages for Home, About, Members, Events, Blog, Projects, and Forum. Data is stored in localStorage; there is no backend.

## Stack
- **Framework:** React 18 + Vite
- **Styling:** Tailwind CSS
- **Routing:** React Router v6
- **Animation:** Framer Motion, Lenis (smooth scroll)
- **Icons:** Lucide React

## Running the app
```
cd client && npm run dev
```
The dev server runs on port 5000 (`vite --host`).

## Structure
```
client/
  src/
    pages/       # Route-level page components
    components/  # Shared UI components
    context/     # React context providers
    data/        # Static/local data
```
